| 所属分组 | 名称              | 必须 | 类型     | 说明                           | 补充       |
| -------- | ----------------- | ---- | -------- | ------------------------------ | ---------- |
| `ws`     | `host`            | 是   | `string` | ws 连接的 host                 | 无         |
| `ws`     | `port`            | 是   | `int`    | ws 连接的 port                 | 无         |
| `ws`     | `interval_time`   | 是   | `int`    | ws 重连尝试时间间隔            | 无         |
| `ws`     | `limit_connect`   | 否   | `int`    | ws 重连尝试次数，-1 为无限重试 | 默认为`-1` |
| `ws`     | `heart_beat_time` | 否   | `int`    | ws 心跳检测中间间隔时间        | 默认为`5`  |
