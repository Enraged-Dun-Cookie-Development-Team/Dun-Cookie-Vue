| 所属分组 | 名称            | 必须 | 类型     | 说明                         | 补充     |
| -------- | --------------- | ---- | -------- | ---------------------------- | -------- |
| `ws`     | `host`          | 是   | `string` | ws连接的host                 | 无       |
| `ws`     | `port`          | 是   | `int`    | ws连接的port                 | 无       |
| `ws`     | `interval_time` | 是   | `int`    | ws重连尝试时间间隔           | 无       |
| `ws`     | `limit_connect` | 否   | `int`    | ws重连尝试次数，-1为无限重试 | 默认为-1 |
